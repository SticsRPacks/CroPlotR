List of provided plots

This vignette give a complete list of all the plots that can be created using the package CroPlotR. The plot names given in italic correspond to the type argument of the respective plot function.

Soil

Type thickness.mswc
Description Thickness against maximal water capacity, the colour shows organic nitrogen concentration.
Essential variables saturated_wtr_cap, depth
Optional variables organic_N_conc

Weather

Type limiting.temperatures
Description Number of hot days against number of cold days. The colour and shape show the year and weather station.
Essential variables temp_day_max, temp_day_min
Optional variables station_name, year
