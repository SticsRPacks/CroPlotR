List of provided plots

This vignette give a complete list of all the plots that can be created using the package CroPlotR. The plot names given in italic correspond to the type argument of the respective plot function. This documentation sometimes figures more than one set of essential variables of a plot; in this case any of the sets can be supplied.

Soil

Type thickness.mswc
Description Thickness against maximal soil water capacity (mswc), the colour shows organic nitrogen concentration.
Essential variables By soil: soil_max_wtr_cap, thickness
By layer: layer_max_wtr_cap, layer_thickness
By layer: layer_water_field_cap, layer_water_wilting_pt, layer_water_wilting_pt, layer_thickness
Optional variables organic_N_conc

Weather

Type limiting.temperatures
Description Number of hot days against number of cold days. The colour and shape show the year and weather station.
Essential variables Option 1: temp_day_max, temp_day_min
Option 2: temp_day_mean
Optional variables station_name, year
